# Compteur

Un programme en C++ permettant de gérer un compteur en ligne de commande.

## Fonctionnalités

- Incrémenter le compteur
- Décrémenter le compteur
- Réinitialiser le compteur
- Afficher la valeur actuelle

## Compilation

Avec **g++** :

```bash
g++ -o compteur main.cpp
```

## Exécution

```bash
./compteur
```

## Technologies

- C++
- g++
